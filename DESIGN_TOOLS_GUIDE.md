## 1. FIGMA (Recommended for Professionals)

### Overview
Figma is a cloud-based design tool perfect for creating scalable, professional logos that can be exported as SVG directly.

### Pros ‚úÖ
- **Free tier** available (perfect for one logo project)
- **SVG export** native - perfect for web apps
- **Scalable components** - easily create icon variations
- **Cloud-based** - access from anywhere
- **Community resources** - free templates and icons
- **Collaboration** - if you need feedback from others
- **Developer-friendly** - code generation and inspect tools

### Cons ‚ùå
- **Learning curve** - takes 1-2 hours to get comfortable
- **Feature-heavy** - more than you need for a simple logo
- **Requires account** - email signup required

### How to Use
1. **Sign up** at figma.com (free)
2. **Create new file** ‚Üí "Logo Design"
3. **Set artboards:**
   - 256x256 for icon
   - 512x256 for horizontal
   - 256x512 for stacked
4. **Use shapes tool** (rectangle, circle, path) to build logo
5. **Export as SVG:**
   - Right-click layer ‚Üí Download
   - Choose SVG format
6. **Export as PNG:**
   - Right-click ‚Üí Export
   - Choose PNG 2x for web

### Cost
- **Free:** 1 file for designs
- **Professional:** $12/month for unlimited files
- **Recommendation:** Stick with free tier for one logo

## 2. CANVA (Easiest Option)

### Overview
Canva is the most beginner-friendly design tool with thousands of templates and drag-and-drop interface.

### Pros ‚úÖ
- **Easiest to learn** - intuitive drag-and-drop
- **Pre-made templates** - hundreds of nutrition/app logos
- **Quick results** - create logo in 1-2 hours
- **AI tools** - AI background generator, image enhancer
- **Mobile app** - design on the go
- **Free tier** - good for getting started
- **PNG export** - easy downloads

### Cons ‚ùå
- **SVG export** - requires paid version ($120/year)
- **Limited control** - less precise than professional tools
- **Template-dependent** - less unique if using templates
- **Watermarks** - free tier adds Canva watermark sometimes
- **Not scalable** - better for static images than components

### How to Use
1. **Sign up** at canva.com (free)
2. **Search:** "Logo" or "App Icon"
3. **Filter:** By color (teal, coral preferred)
4. **Customize:**
   - Change colors to your palette
   - Edit text "EatWise"
   - Add elements (leaves, fork, etc.)
5. **Export:**
   - Free: PNG only
   - Pro: Also gets SVG, transparent PNG

### Cost
- **Free:** PNG export, limited elements
- **Canva Pro:** $120/year - PNG, SVG, advanced features
- **Recommendation:** $10/month for one month if you want SVG

### Unique Feature: AI Magic Design
- Describe your logo in text
- AI generates design options
- Great for getting initial concepts

### Resources
- üîó [Canva Logo Design](https://www.canva.com/create/logos/)

## 4. INKSCAPE (Free, Powerful Alternative)

### Overview
Free, open-source vector design software. Professional-grade but steeper learning curve.

### Pros ‚úÖ
- **100% free** - no subscription
- **Open-source** - community-driven development
- **SVG native** - created specifically for SVG
- **Professional quality** - used by professionals worldwide
- **Powerful tools** - Bezier curves, path operations, layers
- **No ads or watermarks**

### Cons ‚ùå
- **Steep learning curve** - 4-8 hours to get comfortable
- **Slower than web tools** - desktop application
- **Less polished UI** - older interface design
- **Less intuitive** - more technical
- **No AI features** - manual design only

### How to Use
1. **Download** from inkscape.org (free)
2. **Learn basics:**
   - Selection tool (arrow)
   - Bezier tool (pen)
   - Shapes (circle, rectangle)
3. **Create logo:**
   - Start with shapes
   - Combine using path operations
   - Add text with text tool
4. **Export:**
   - File ‚Üí Save As ‚Üí SVG
   - Also export PNG for preview

### Cost
- **Free** - always

### Best for
- If you want full control and have time
- If you're comfortable with design software
- Long-term projects with future updates

### Resources
- üîó [Inkscape Home](https://inkscape.org/)
- üìö [Official Tutorials](https://inkscape.org/learn/)

## Export Settings Guide

### **For SVG (Best for Web)**
‚úÖ **Figma:**
- Right-click layer ‚Üí Download
- Format: SVG
- ‚úì Include "text outlines"

‚úÖ **Adobe Express:**
- Download ‚Üí SVG format
- Keep vector format

‚úÖ **Canva Pro:**
- Download ‚Üí SVG format

‚úÖ **Inkscape:**
- File ‚Üí Save As ‚Üí Scalable Vector Graphic (*.svg)

### **For PNG (Backup/Preview)**
- Export at 256px width (72 DPI)
- Save with transparent background
- 2x version (512px) for high-quality displays

### **For PNG (Print)**
- Export at 600px+ width (300 DPI)
- Save with transparent background for versatility

## Final Checklist Before Exporting

- [ ] Logo is recognizable at 16px size
- [ ] All text is outlined (not dependent on fonts)
- [ ] Colors match hex codes (#10A19D, #52C4B8, #FF6B6B)
- [ ] SVG file size under 50KB
- [ ] Transparent background (no white background)
- [ ] Saved in `/assets` folder with proper naming
- [ ] Multiple size versions created (if PNG)
