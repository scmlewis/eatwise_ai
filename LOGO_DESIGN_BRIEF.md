# EatWise Logo Design Brief

## Project Overview
**Brand Name:** EatWise  
**Tagline:** "Your AI-Powered Nutrition Hub"  
**Industry:** Health & Nutrition Technology / Meal Tracking App  
**Target Audience:** Health-conscious individuals aged 18-50, nutrition enthusiasts, fitness-focused users  
**Platform:** Web & Mobile App (Streamlit-based)

---

## Brand Story & Vision

EatWise is an intelligent nutrition assistant that transforms how people track and understand their eating habits. Using AI-powered analysis, the app provides:
- Smart meal logging (text or photo-based)
- Instant nutritional breakdown
- Personalized health recommendations
- Gamified progress tracking with badges and streaks

The brand should feel **modern, trustworthy, and motivating** ‚Äî encouraging users to make better nutrition choices without judgment.

## Logo Variations Required

### 1. **Full Logo (Horizontal)**
- Logo + Brand Name "EatWise"
- Use: Website header, marketing materials, splash screens
- Minimum width: 200px
- Proportions: ~3:1 width to height ratio

### 2. **Full Logo (Stacked)**
- Logo above/around Brand Name
- Use: App sidebar, mobile headers
- Minimum width: 100px
- Proportions: ~1:1 to 2:1

### 3. **Icon/Mark Only**
- Just the symbol without text
- Sizes: 16px, 32px, 64px, 128px, 256px, 512px
- Use: Favicon, app icon, badges, avatars
- Minimum size: 16px should be recognizable

### 4. **Monochrome/Single Color**
- Black and white versions
- Use: Print, embroidery, restricted color environments
- Dark version on light background & light version on dark background

### 5. **Outlined/Line Version**
- Stroke-based design for minimal backgrounds
- Use: Light mode interfaces, line icon sets
- Stroke width: 2px

## Design Specifications

### Technical Requirements
- **Format:** SVG (scalable vector) - PRIMARY
- **Backup Formats:** PNG, PDF, EPS
- **Resolution (PNG):** 300x300px minimum (72 DPI for screen, 300 DPI for print)
- **File Naming Convention:**
  - `eatwise-logo-full-horizontal.svg`
  - `eatwise-logo-stacked.svg`
  - `eatwise-icon.svg`
  - `eatwise-icon-monochrome.svg`
  - `eatwise-logo-outlined.svg`

### Scalability Requirements
- Logo must be recognizable at 16px and larger
- No fine details that disappear at small sizes
- Clear at all breakpoints (mobile, tablet, desktop)

### Color Specifications
- Use HEX codes for digital delivery
- Include CMYK values for print (if applicable)
- Provide color swatches for implementation

## Design Process Recommendations

### For Design Tools (Figma, Adobe XD, Canva)
1. Start with grid/geometry to ensure proportions
2. Create symbol/shape primitives
3. Build logo from components (easier to iterate)
4. Test at multiple sizes (16px, 32px, 64px, 256px)
5. Export SVG with proper settings (no raster effects)
6. Convert to PNG at 300px width for web preview

### For Hiring a Designer
1. Share this brief in full
2. Request 3-5 logo concepts
3. Specify revision rounds (usually 2-3)
4. Request all formats and sizes
5. Get intellectual property rights assigned to you
6. Timeline: 1-2 weeks typical

---

## Questions for Designer/Design Tool

Before starting, clarify:
1. Should the logo include geometric patterns or stay minimal?
2. Prefer leaf/fork emphasis or AI/tech emphasis?
3. Should logo work in gradient or solid colors only?
4. Any animal/character elements or symbol-based only?
5. How much of the design should be "negative space" based?
